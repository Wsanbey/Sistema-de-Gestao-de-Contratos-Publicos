# Sistema de Gestao de Contratos Publicos

## Descricao
Painel web para cadastro, controle e monitoramento de contratos publicos com aditivos, alertas automaticos e relatorios gerenciais.

## Funcionalidades
- Cadastro completo de contratos
- Controle de aditivos e alteracoes
- Alertas automaticos para vencimentos
- Relatorios gerenciais
- Dashboard com metricas

## Tecnologias
- Frontend: Vue.js
- Backend: JavaScript/Node.js
